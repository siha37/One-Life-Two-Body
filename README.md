# ARM BODY 프로젝트

유니티 기반의 탑다운 뱀서류 게임 프로젝트입니다. 
한명은 이동을
다른 한명은 공격을 담당하며 플레이 합니다.
액티브 및 패시브 스킬 시스템, 레벨업을 통한 성장, 그리고 데이터 기반의 게임 설계를 특징으로 합니다.

## 주요 시스템

### 1. 스킬 시스템 및 발사체 처리
스킬 시스템은 `TOP_Projectile` 클래스를 기본으로 하며, 액티브(P2)와 패시브(P1) 스킬로 나뉩니다.

- 발사 방식(ShotType): Click, ClickAndHold, Hold, Passive 등 다양한 발사 로직을 지원합니다.
- 자원 관리: 스킬 사용 시 MANA 또는 ENERGY를 소모하며, `P_Status`를 통해 실시간으로 계산됩니다.
- 발사체 구현: `BulletBasic`, `P_ShotGun`, `P_LazzerSniper`, `P_Sowrd` 등 개별 클래스에서 고유의 발사 로직과 충돌 처리를 담당합니다.
- 데이터 연동: 각 스킬은 공격력, 속도, 관통 횟수, 발사체 개수 등의 능력치를 레벨별로 관리합니다.

### 2. 레벨업 시스템
플레이어의 성장은 `Level_UP` 클래스에서 관리하며, 전략적인 선택 요소를 제공합니다.

- 스킬 선택 로직: 레벨업 시 액티브 스킬과 패시브 스킬 중 하나를 번갈아 가며 선택할 수 있는 구조입니다.
- 무작위 스킬 추출: 현재 보유한 스킬 중 최대 레벨에 도달하지 않은 스킬들을 무작위로 추출하여 UI에 표시합니다.
- 부가 효과: 스킬 선택 시 체력 회복 및 일시적인 무적 상태를 부여하여 게임 플레이의 안정성을 높입니다.

### 3. CSV 데이터 처리
게임 내 다양한 수치 데이터는 `CSVReader`를 통해 효율적으로 관리됩니다.

- 데이터 파싱: 정규식을 사용하여 CSV 파일을 읽고, 이를 `Dictionary<string, object>` 형태로 변환합니다.
- 유연한 확장성: 적 스폰 데이터, 스킬 능력치 데이터 등을 외부 CSV 파일에서 관리하여 코드 수정 없이 밸런스 조절이 가능합니다.

### 4. 적군 오브젝트 생성 처리
적의 생성은 `EnemySpawn` 클래스에서 시간대별, 패턴별로 관리됩니다.

- 타이머 연동: `InGameTimer`의 현재 시간을 기준으로 미리 정의된 스폰 데이터에 맞춰 적을 생성합니다.
- 다양한 스폰 패턴:
    - Random: 화면 경계면에서 무작위 생성
    - DirectionRandom: 상, 하, 좌, 우 특정 방향에서 생성
    - Clockwise / CounterClockwise: 플레이어를 중심으로 회전하며 생성
- 객체 관리: 생성된 적들은 `EnemyCollection` 하위에서 관리되며, 다음 스폰 단계로 넘어갈 때 이전 적들을 별도로 분리하여 최적화합니다.

## 기술 스택
- Engine: Unity
- Language: C#
- Animation: Spine
- Data Format: CSV, JSON (Newtonsoft.Json)
